Le système de score au soft tennis diffère de celui du tennis traditionnel, utilisant un système unique. Voici les principales caractéristiques et règles du score au soft tennis :

## Règles de base du score

1. **Format du match** :
   - Les matchs en simple se jouent généralement au meilleur des trois sets.
   - Les matchs en double se jouent généralement au meilleur des cinq sets.

2. **Score par set** :
   - Chaque set se joue en 7 points.
   - Le premier à atteindre 7 points remporte le set.

3. **Jeu décisif** :
   - Si le score atteint 6-6, un jeu décisif en 7 points est joué.
   - Dans le jeu décisif, le premier à atteindre 7 points avec deux points d'écart remporte le set.

## Détails du score

1. **Comment marquer des points** :
   - Après chaque échange, le gagnant marque 1 point.
   - Le serveur et le receveur peuvent tous deux marquer des points.

2. **Changement de service** :
   - Le service change après chaque point.
   - Dans le jeu décisif, le service change tous les 2 points.

   - Au soft tennis, il n'y a qu'une seule tentative de service.

## Détermination du vainqueur

1. **En simple** :
   - Au meilleur des trois sets, le joueur qui remporte deux sets gagne le match.

2. **En double** :
   - Au meilleur des cinq sets, l'équipe qui remporte trois sets gagne le match.

## Affichage du score

Dans un match de soft tennis, le tableau de score affiche généralement :

- Le nombre de sets actuel
- Le score de chaque set
- L'indication du serveur

## Principales différences avec le tennis traditionnel

1. **Système de score** :
   - Le soft tennis utilise un système à 7 points.
   - Le tennis traditionnel utilise le système 15-30-40.

2. **Tentatives de service** :
   - Le soft tennis n'autorise qu'une seule tentative de service.
   - Le tennis traditionnel permet deux tentatives de service.

3. **Durée du match** :
   - Les matchs de soft tennis sont généralement plus courts.
   - Les matchs de tennis traditionnel peuvent être plus longs, surtout en format cinq sets.

Comprendre le système de score du soft tennis est essentiel pour apprécier et participer à ce sport. Ce système de score unique rend le jeu plus rapide et plus excitant, tout en exigeant une concentration élevée des joueurs à chaque échange.
