# 擊球動作

本章節將介紹軟式網球的基本擊球動作，包括正手擊球、反手擊球以及一些常見錯誤和改正方法。掌握這些基本技巧對於提高您的軟式網球水平至關重要。

## 正手擊球

正手擊球是軟式網球中最常用的技巧之一。以下是正確執行正手擊球的步驟：

1. **準備姿勢**：側身面對來球，重心略微前傾。
2. **後搖**：將球拍向後拉，手臂自然彎曲。
3. **擊球點**：在身體側前方擊球，保持手臂伸直。
4. **跟隨動作**：擊球後，球拍自然向前上方揮動。

## 反手擊球

反手擊球雖然不如正手常用，但同樣重要。以下是反手擊球的基本步驟：

1. **準備姿勢**：側身面對來球，非持拍手輕扶球拍頭。
2. **後搖**：將球拍拉至身體另一側。
3. **擊球點**：在身體前方擊球，手臂自然伸展。
4. **跟隨動作**：擊球後，球拍向目標方向揮動。

## 常見錯誤和改正方法

1. **擊球點過遠或過近**
   - 錯誤：導致力量不足或控制不佳
   - 改正：練習判斷來球軌跡，調整站位

2. **手腕過度使用**
   - 錯誤：影響擊球穩定性
   - 改正：保持手腕穩定，主要使用手臂力量

3. **重心不穩**
   - 錯誤：影響擊球質量和移動速度
   - 改正：注意腳步移動，保持重心穩定

4. **視線離開球**
   - 錯誤：無法準確判斷擊球時機
   - 改正：保持視線跟隨球的軌跡直到擊球

通過持續練習和糾正這些常見錯誤，您將能夠逐步提高軟式網球的擊球技巧。記住，良好的基本功是成為優秀軟式網球選手的關鍵。
