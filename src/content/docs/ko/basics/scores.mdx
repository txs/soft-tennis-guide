소프트 테니스의 점수 체계는 하드 테니스와 다르며, 독특한 시스템을 채택하고 있습니다. 다음은 소프트 테니스 점수의 주요 특징과 규칙입니다:

## 기본 점수 규칙

1. **경기 형식**:
   - 단식 경기는 일반적으로 3전 2승제로 진행됩니다.
   - 복식 경기는 일반적으로 5전 3승제로 진행됩니다.

2. **각 게임 점수**:
   - 각 게임은 7점제로 진행됩니다.
   - 먼저 7점을 얻은 쪽이 해당 게임의 승자가 됩니다.

3. **타이브레이크**:
   - 점수가 6-6이 되면 7점 타이브레이크를 진행합니다.
   - 타이브레이크에서 먼저 7점을 얻고 상대보다 2점 앞선 쪽이 승리합니다.

## 점수 세부 사항

1. **점수 획득 방법**:
   - 각 랠리가 끝난 후 승리한 쪽이 1점을 얻습니다.
   - 서비스하는 쪽이나 받아치는 쪽 모두 점수를 얻을 수 있습니다.

2. **서비스 교체**:
   - 1점을 얻을 때마다 서비스가 교체됩니다.
   - 타이브레이크에서는 2점마다 서비스가 교체됩니다.

3. **코트 교체**:
   - 각 게임이 끝난 후 쌍방이 코트를 교체합니다.
   - 타이브레이크에서는 4점마다 코트를 교체합니다.

4. **서브 실수**:
   - 소프트 테니스는 서브 기회가 1번만 주어집니다.
   - 서브 실수는 즉시 상대에게 점수를 줍니다.

## 경기 승패 판별

1. **단식 경기**:
   - 3전 2승제로, 먼저 2게임을 이긴 선수가 승리합니다.

2. **복식 경기**:
   - 5전 3승제로, 먼저 3게임을 이긴 팀이 승리합니다.

## 점수판 표시

소프트 테니스 경기에서 점수판에는 일반적으로 다음과 같은 정보가 표시됩니다:

- 현재 게임 수
- 각 게임의 점수 상황
- 서비스하는 쪽의 표시

## 하드 테니스와의 주요 차이점

1. **점수 체계**:
   - 소프트 테니스는 7점제를 사용합니다.
   - 하드 테니스는 15-30-40 점수 체계를 사용합니다.

2. **서브 기회**:
   - 소프트 테니스는 서브 기회가 1번만 주어집니다.
   - 하드 테니스는 2번의 서브 기회가 주어집니다.

3. **경기 길이**:
   - 소프트 테니스 경기는 보통 더 짧습니다.
   - 하드 테니스 경기는 특히 5세트 경기에서는 더 길어질 수 있습니다.

소프트 테니스의 점수 체계를 이해하는 것은 이 스포츠를 감상하고 참여하는 데 매우 중요합니다. 이러한 독특한 점수 시스템은 경기의 속도를 더 빠르고 자극적으로 만들어 주며, 선수들에게 각 랠리에서 집중력을 유지할 것을 요구합니다.
