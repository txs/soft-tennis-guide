---
title: Sistema de puntuación del tenis suave
description: Comprende el sistema único de puntuación y las reglas de juego del tenis suave.
---

El sistema de puntuación del tenis suave difiere del tenis duro, utilizando un sistema único. Aquí están las principales características y reglas de puntuación del tenis suave:

## Reglas básicas de puntuación

1. **Formato del partido**:
   - Los partidos individuales generalmente se juegan al mejor de tres sets.
   - Los partidos de dobles generalmente se juegan al mejor de cinco sets.

2. **Puntuación por set**:
   - Cada set se juega a 7 puntos.
   - El primero en alcanzar 7 puntos gana el set.

3. **Set decisivo**:
   - Si el marcador llega a 6-6, se juega un set decisivo a 7 puntos.
   - En el set decisivo, el primero en alcanzar 7 puntos y tener una ventaja de 2 puntos gana.

## Detalles de puntuación

1. **Cómo se ganan puntos**:
   - Después de cada rally, el ganador obtiene 1 punto.
   - Tanto el servidor como el receptor pueden ganar puntos.

2. **Cambio de servicio**:
   - El servicio cambia después de cada punto.
   - En el set decisivo, el servicio cambia cada 2 puntos.

3. **Cambio de lado**:
   - Los jugadores cambian de lado después de cada set.
   - En el set decisivo, los jugadores cambian de lado cada 4 puntos.

4. **Faltas de servicio**:
   - En el tenis suave, solo hay una oportunidad de servicio.
   - Una falta de servicio resulta en un punto directo para el oponente.

## Determinación del ganador del partido

1. **Partidos individuales**:
   - Al mejor de tres sets, el jugador que gane dos sets gana el partido.

2. **Partidos de dobles**:
   - Al mejor de cinco sets, el equipo que gane tres sets gana el partido.

## Visualización del marcador

En los partidos de tenis suave, el marcador generalmente muestra:

- El número de set actual
- La puntuación de cada set
- Indicador del servidor

## Principales diferencias con el tenis duro

1. **Sistema de puntuación**:
   - El tenis suave utiliza un sistema de 7 puntos.
   - El tenis duro utiliza el sistema de puntuación 15-30-40.

2. **Oportunidades de servicio**:
   - En el tenis suave, solo hay una oportunidad de servicio.
   - En el tenis duro, hay dos oportunidades de servicio.

3. **Duración del partido**:
   - Los partidos de tenis suave suelen ser más cortos.
   - Los partidos de tenis duro pueden ser más largos, especialmente en partidos al mejor de cinco sets.

Entender el sistema de puntuación del tenis suave es crucial para apreciar y participar en este deporte. Este sistema único de puntuación hace que los partidos sean más rápidos y emocionantes, a la vez que exige a los jugadores mantener una alta concentración en cada punto.
